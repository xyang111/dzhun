## 一、背景与目标

贷准（dzhun.com.cn）是一款 C 端 AI 征信分析产品。用户上传征信报告后，系统提供有效贷款/信用卡数据、月供、负债、信用卡使用率等指标展示，102维评分，问题诊断，优化方案及匹配产品。

**宣传图目标：** 面向 C 端引流，覆盖4类用户（迷茫型/优化型/修复型/小白型），多渠道通用（微信朋友圈、代理商私域、抖音、小红书等）。

**用户行动路径（双 CTA）：**
1. 扫码进入官网，立即免费体检
2. 扫码关注微信服务号，获取专属方案

---

## 二、核心创意概念

### "征信体检" 医生视角

**类比逻辑：**
- 贷准 = 医生
- 用户征信 = 病人/体检对象
- 服务 = 全面体检 + 精准诊断 + 改善处方

**优势：**
- 去污名化：不是"你征信差"，而是"人人都该查一查"
- 降低心理门槛：体检是常识，不是危机
- 场景化强，覆盖所有4类用户
- 与竞品差异化显著（竞品多为"贷款秒下/低息"硬广）

## 七、执行方式

### 方案 A：设计软件出图（推荐）
- 工具：Figma / Canva / Photoshop
- 按上述图层结构逐层还原
- 输出 PNG，分辨率 72dpi（屏显）/ 300dpi（印刷备用）

### 方案 B：HTML/CSS 生成图（快速迭代用）
- 按 mockup 代码还原为静态 HTML
- 用浏览器截图工具（如 puppeteer）导出 PNG
- 适合快速验证文案和布局，后期再由设计师精修

## 八、后续延伸

- **系列图：** 按4类用户分别出定制版（如"贷款被拒版"/"利率太高版"），主标题换症状，底部保持统一
- **A/B测试：** 主标题 A（疑问句）vs C（很多人...）对比点击率
- **动态版：** 症状红点依次出现动效，适合抖音/视频号
- **操作指引图（独立素材）：** 单独出一张"如何下载简版征信"步骤图，作为配套第二张素材。投放逻辑：宣传图引发兴趣 → 用户问"怎么查" → 发送指引图。放置位置：网站首页上传区下方说明文案、微信服务号欢迎消息。不放入主宣传图，避免信息过载。
