## Debug Skill

在写任何代码之前，按以下顺序诊断问题，**每一步得出结论后再进入下一步**：

1. **线上代码是最新版吗？**
   - 检查 `index.html` 中的 `?v=` 版本号是否与最近部署一致
   - 检查 Cloudflare Worker 的 Version ID 是否是最新部署的那个
   - CEO 是否在 Dashboard 上直接改过 Worker？

2. **是缓存/CDN 问题吗？**
   - 强刷页面（Cmd+Shift+R）后问题还在吗？
   - curl 直接请求 API 端点，绕过浏览器缓存验证

3. **是环境问题吗？**
   - 是否只在特定设备/浏览器/微信内出现？
   - 是 HTTP vs HTTPS 问题吗？
   - 是微信内置浏览器的特殊限制吗？

4. **前端和后端有冲突的硬编码值吗？**
   - 检查前端是否有覆盖 Worker 设置的硬编码参数（如 `max_tokens`、`model`）
   - Worker 改动后前端请求参数是否同步更新了？

**确认根因后，用一句话说明根因，再提出最小化修复方案。**
